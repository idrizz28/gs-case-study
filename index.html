Edge Case Coverage

Built for the hard cases.

Every scenario that would normally drop a lead or require staff is handled in-flow.

👤
New adult lead
Full qualification → data capture → calendar booking, any time of day.
👦
Parent enrolling a child
Youth path with age-routing to the correct kids calendar. Parent info also captured.
👨‍👦
Parent + self enrolling
Multi-enrollee: both bookings completed in a single session.
🚫
Under-18 self-booking
Minor gate fires → captures info → routes to team for guardian follow-up.
💬
Pricing question
Firm redirect to trial every time. No dollar figures. Ever.
📋
Non-bookable program
Acknowledged without a booking path. Redirected to trial for in-person context.
🤝

One tag disables the bot instantly on any conversation. Never re-engages.
⚡
Hostile / opt-out
Graceful de-escalation. No booking attempt. No argument.
